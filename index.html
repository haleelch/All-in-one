How to Add This Website to Google Search

To make your website searchable on Google, follow these simple steps:

  1. Go to Google Search Console and sign in.
  2. Select URL prefix and enter your Vercel website link (e.g., https://your-project.vercel.app).
  3. Choose the HTML Tag verification method.
  4. Copy the verification code provided by Google (the long alphanumeric code inside content="...").
  5. Edit your Github index.html file, find line 17, and replace YOUR_GOOGLE_VERIFICATION_CODE_HERE with your code.
  6. Commit the changes. Once Vercel deploys it, click Verify in Google Search Console!
